Website Gift Wrap/Packing Odoo Apps

This Odoo apps helps for Customer to Add Gift Wrap on their Product in Odoo Website. With using this Odoo apps helps website users/visitors can easily select wrap packing on website for purchase products.

Features

Gift Wrap Product Configuration

You can configure product and product price for website gift wrap.

Enable Disable Gift Wrap

You can easily enable and disable gift wrap on website.

Add Gift Wrap Note Dynamically on Website

User can add gift wrap note dynamically on website.

Add Gift Wrap Product in Cart

User can add gift wrap product in shopping cart.

Remove Gift Wrap Product

User can easily remove gift wrap product from Shopping Cart.

Gift Wrap Product in Sale Order

User can see gift wrap product in sale order line.
Demo Request Video Tutorial

Create Gift Wrap and List Of Gift Wrap

User can create multiple gift wrap.

User can see created multiple gift wrap.

Gift Wrap Configuration

Configure Gift Wrap Configuration Goto Website Admin > GiftWrap Configuration.Manager can configure product and product price for website gift wrap.

User can see listed multiple Gift Wrap Configuration.

Enable/Disable Gift Wrap

On website shopping cart view you can enable/disable the gift wrap functionality.

Add Gift Wrap with Note

After enable gift wrap, You can see gift wrap section in website cart.
Here you can apply gift wrap by entering note for product.

Added Gift Wrap Product

After applying gift wrap, gift wrap product will be added in cart.
You can remove gift wrap product from cart by clicking remove button.

After submit the gift wrap with product, then now use can pay payment with gift wrap.

Order is successfully done.

Sale Order View in Back-end

Here you can see gift wrap product will be added in sale order with defined description and product price.
