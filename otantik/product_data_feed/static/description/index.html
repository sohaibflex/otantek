Base module to generate product data feeds

Description

Many systems need a product source to get product data from Odoo. This module provides this engine to get product data source by the URL.

The Odoo Product Data Feed app family by Garazd Creation provides a usable tool to integrate with analytic systems and marketplaces such as Google Merchant Center | Google Shopping Feed, Pinterest Catalog, Facebook Catalog and Instagram Shopping.
